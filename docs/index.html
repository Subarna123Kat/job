Connect Your GitHub Repo

This dashboard manages your job application data directly in your private GitHub repository. One-time setup — takes 2 minutes.

  1. Create a private GitHub repository and push this project to it.
  2. Go to github.com/settings/tokens/new → Generate a Classic PAT with repo + workflow scopes.
  3. In your repo → Settings → Secrets → Actions, add GMAIL_USER and GMAIL_APP_PASSWORD.
Your PAT is stored only in this browser (localStorage). Never shared.

Why is it stored in GitHub?

Your CV is committed directly to your private repository as cv/resume.pdf. When GitHub Actions runs, it reads the CV from the repo and attaches it to every email. This means nothing is stored on any external server — it's all yours.
